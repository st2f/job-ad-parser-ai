- Salary: 65k€ - 75k€
- Work model: hybrid

## Stack

- Python

- Docker

- ECS

## Summary

RealTekno recherche un Senior Backend Engineer Python pour renforcer son équipe produit/tech sur une plateforme SaaS dédiée au pilotage de la consommation et des opérations dans l’énergie. Le poste est basé à Paris avec une organisation flexible et du télétravail la plupart du temps, mais avec des présences ponctuelles sur site pour certains ateliers et réunions. Le rôle couvre l’évolution du backend, la conception d’API, l’architecture, la performance et la fiabilité, dans une équipe à taille humaine.

## Requirements

- Idéalement 5 ans ou plus d'expérience en backend
- Bonne maîtrise de Python, ou à défaut un langage serveur proche avec une vraie capacité à monter rapidement
- Être solide sur les API REST
- Être solide sur les systèmes distribués
- Être solide sur les bases SQL
- Bonne communication
- Capacité à expliquer simplement ses choix
- Envie de faire progresser le collectif
- Être OK pour venir ponctuellement au bureau à Paris

## Nice to have

- Sujets infra / cloud AWS
- Pipelines de données ou ETL
- Un peu de frontend, typiquement React

## Responsibilities

- Faire évoluer des services backend existants
- Corriger ce qui doit l'être
- Concevoir de nouvelles briques API
- Reprendre certains sujets d'architecture
- Travailler la performance, l'observabilité et la fiabilité
- Collaborer avec les personnes côté front / data
- Revoir du code et participer aux discussions techniques
- Contribuer aux choix techniques quand il faut trancher

## Benefits

- Fonctionnement flexible avec télétravail
- 10 RTT
- Mutuelle prise en charge à 100%
- Retraite / offsite d'équipe une fois par an
- Environnement où on peut proposer des idées et avoir de l'impact

## Hiring process

Premier échange RH en visio, puis entretien technique, puis discussion finale avec le CTO.
